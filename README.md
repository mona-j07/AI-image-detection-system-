
🛰 MirageX
AI-Generated Image Detection System

MirageX is a deep learning–based image forensics system designed to detect whether an image is AI-generated or real.

As synthetic media becomes increasingly realistic, MirageX helps identify hidden artifacts, frequency inconsistencies, and structural anomalies left behind by generative models.

## Run Locally

**Prerequisites:**  
Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
